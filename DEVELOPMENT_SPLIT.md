# 开发分工

当前只记录开发任务，不包含课程设计报告和答辩材料。

| 成员 | 占比 | 后端开发内容 |
| --- | ---: | --- |
| 组长 | 70% | 项目骨架、数据库主设计、统一响应、异常处理、Session 登录鉴权、用户接口、订单创建、库存扣减、模拟支付、后台权限拦截、整体联调 |
| 组员A | 10% | 分类查询、商品分页、商品详情、商品搜索接口 |
| 组员B | 10% | 购物车增删改查、勾选、登录后购物车合并、收货地址 CRUD、默认地址 |
| 组员C | 10% | 后台商品管理、分类管理、订单查询、发货状态修改 |

建议分支：

- 组长：`feature/core-flow`
- 组员A：`feature/product-catalog`
- 组员B：`feature/cart-address`
- 组员C：`feature/admin-manage`
